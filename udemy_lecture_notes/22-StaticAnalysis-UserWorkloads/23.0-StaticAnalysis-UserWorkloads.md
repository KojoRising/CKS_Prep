## 2 Kubesesc
- Security Risk Analysis - For K8s resources
- Opensource
- Opinionated - Fixed Set of Rules (Security Best Practices)
- Run as:
  - Binary
  - Docker Container
  - Kubectl Plugin
  - Admission Controller (kubesec-webhook)
  
`docker run -i kubesec/kubesec:512c5e0 scan /dev/stdin < kubesec-test.yaml`
